Доступность и SEO

Настройки для комфортного использования сайта и информация для поисковых систем

Возможности доступности

Размер шрифта

Увеличивайте текст до комфортного размера. Доступны три варианта: обычный, увеличенный и крупный.

  • Обычный: 16px
  • Увеличенный: 20px (+25%)
  • Крупный: 24px (+50%)

Высокая контрастность

Режим с максимальным контрастом для людей с нарушениями зрения. Чёрный фон, белый текст, жёлтые акценты.

  • Улучшает читаемость
  • Соответствует стандарту WCAG AAA
  • Подходит для слабовидящих

Режим для дальтоников

Специальный фильтр для людей с нарушением цветовосприятия (протанопия, дейтеранопия).

  • Адаптирует цвета
  • Улучшает различимость
  • Сохраняет читаемость

Озвучивание текста

Функция чтения текста вслух с помощью встроенного синтезатора речи браузера.

  • Читает выделенный текст
  • Поддержка русского языка
  • Регулировка скорости

Навигация с клавиатуры

Полная поддержка навигации без мыши. Используйте Tab, Enter, Escape для управления.

  • Tab - переход между элементами
  • Enter - активация
  • Escape - закрытие

Отключение анимаций

Для людей с вестибулярными нарушениями или предпочитающих статичный интерфейс.

  • Убирает все анимации
  • Сохраняет функциональность
  • Ускоряет загрузку

SEO-оптимизация

Реализованные SEO-функции

Мета-теги

title, description, keywords, robots, canonical URL, Open Graph для социальных сетей

Schema.org разметка

Структурированные данные для поисковых систем (EducationalOrganization)

Семантическая HTML5 разметка

Использование header, nav, main, section, article, footer, aside

ARIA-атрибуты

role, aria-label, aria-expanded, aria-current, aria-hidden для скринридеров

Адаптивный дизайн

Mobile-first подход, viewport meta tag, медиа-запросы

Оптимизация скорости

Минимизация CSS, lazy loading изображений, кэширование

HTTPS и безопасность

SSL-сертификат, безопасные соединения

XML Sitemap

Карта сайта для индексации поисковыми системами

Robots.txt

Файл с инструкциями для поисковых роботов

Альтернативный текст для изображений

Описание всех изображений для скринридеров и SEO

📋 Соответствие стандартам WCAG 2.1

Воспринимаемость

  • Текстовые альтернативы для нетекстового контента
  • Альтернативы на основе времени (субтитры)
  • Адаптируемость контента
  • Различимость (контраст 4.5:1)

Управление

  • Доступность с клавиатуры
  • Достаточное время для чтения
  • Защита от судорог (не более 3 миганий/сек)
  • Навигационные механизмы

Понятность

  • Читаемость текста
  • Предсказуемость интерфейса
  • Помощь при вводе данных
  • Ясный язык контента

Совместимость

  • Совместимость со скринридерами
  • Валидный HTML/CSS
  • Семантическая разметка
  • ARIA-атрибуты

💡 Рекомендации по улучшению

Изображения

Добавьте alt-текст ко всем изображениям. Используйте описательные имена файлов (не IMG_1234.jpg, а students-in-lab.jpg).

Внутренние ссылки

Создайте перекрёстные ссылки между страницами. Используйте описательные анкоры вместо "кликните здесь".

Мобильная версия

Тестируйте на реальных устройствах. Используйте Google Mobile-Friendly Test для проверки.

Скорость загрузки

Оптимизируйте изображения (WebP формат), используйте CDN, включите кэширование браузера.

Контент

Регулярно обновляйте контент. Создавайте блог с полезными статьями для абитуриентов.

Мультиязычность

Добавьте английскую версию сайта для иностранных абитуриентов. Используйте hreflang теги.
